# TechFlow Solutions - Gerenciador de Tarefas Ágil

## 🎯 Objetivo do Projeto
Sistema de gerenciamento de tarefas desenvolvido para uma startup de logística, focado em acompanhar o fluxo de trabalho em tempo real, priorizar tarefas críticas e monitorar o desempenho da equipe.

## ⚙️ Metodologia Adotada
O projeto utiliza **Kanban** para a gestão visual do fluxo de trabalho, garantindo entregas contínuas e flexibilidade para mudanças de escopo. A qualidade do código é garantida através de Integração Contínua (CI) com testes automatizados.

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Python
- **Testes:** PyTest
- **CI/CD:** GitHub Actions
